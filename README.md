# blog-vue-typescript

## 项目结构

```
├── public                          // 静态页面
├── scripts                         // 相关脚本配置
├── src                             // 主目录
    ├── assets                      // 静态资源
    ├── components                  // 组件
    ├── lib                         // 全局插件
    ├── store                       // store
    ├── utils                       // 工具方法
    ├── styles                      // 样式
    ├── views                       // 页面
    ├── App.vue                     // 页面主入口
    ├── main.ts                     // 脚本主入口
    ├── router.ts                   // 路由配置
├── tests                           // 测试用例
├── package.json                    // 依赖
├── tsconfig.json                   // ts 配置
└── tslint.json                     // tslint 配置

```

```
npm install
```
